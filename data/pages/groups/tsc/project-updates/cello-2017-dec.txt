===== Project Health =====
Overall, the project is going as planned: 
  - Active discussions happen at rocket channel; 
  - Most patchsets get response within 24 hours; 
  - New users are coming to adopt cello and report demands/bugs.


===== Issues =====
There are no issues at this time.

Release history:

We are preparing the v0.8.0 release at end of Dec, 2017.

===== Overall Activity in the Past Quarter =====
Most questions happen at rocket chat, and get response within a day. New features are proposed and planed using jira, also with fixing reported bugs.

===== Current Plans =====
v0.8.0 release will come soon, with supports of setting up fabric network using ansible, on top of cloud env, vm or servers. The vsphere cluster support will be done, too. Moreover, the documentation keeps updated with the latest design and implementation.

Besides, we are planning to have some meetup recently.
