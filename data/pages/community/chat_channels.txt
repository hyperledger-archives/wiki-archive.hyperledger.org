====== Channel Guide for Hyperledger's Rocket Chat ======

<WRAP center round box 75%>
NOTE: This page is no longer maintained. If you want a full list of chat channels, please visit https://chat.hyperledger.org/directory.
</WRAP>

Hyperledger projects use RocketChat to conduct ephemeral discussions; whereas, mailing lists are used to conduct larger discussions for which we want to maintain a history.  To join our chat discussions, you will need a [[https://identity.linuxfoundation.org/|Linux Foundation ID]] to login to [[https://chat.hyperledger.org/]].

  * [[https://chat.hyperledger.org/channel/ambassadors|#ambassadors]]\\ Used by the [[community:ambassador |Technical Ambassadors]] to communicate amongst themselves and with others who are interested in having a technical ambassador speak at an event

  * [[https://chat.hyperledger.org/channel/architecture-wg|#architecture-wg]]\\ Used by the [[groups:architecture:architecture-wg|Architecture Working Group]] to announce meeting updates and adhoc meetings

  * [[https://chat.hyperledger.org/channel/blockchain4good|#blockchain4good]]\\ Used for discussions for the participants of the 2017 Dublin Blockchain for Good Hackathon held at Accenture The Dock 

  * [[https://chat.hyperledger.org/channel/burrow|#burrow]]\\ Used by the Hyperledger Burrow project to discuss creating a permissioned smart contract machine based on the Ethereum Virtual Machine (EVM) specification

  * [[https://chat.hyperledger.org/channel/burrow-contributors|#burrow-contributors]]\\ Used for discussions by the contributors/maintainers of the Hyperledger Burrow project

  * [[https://chat.hyperledger.org/channel/caliper|#caliper]]\\ Used by the Hyperledger Caliper project for questions and ephemeral discussions

  * [[https://chat.hyperledger.org/channel/caliper-contributors|#caliper-contributors]]\\ Used by the Hyperledger Caliper project to discuss implementation details of the project

  * [[https://chat.hyperledger.org/channel/cello|#cello]]\\ Used by the Hyperledger Cello project to discuss bringing an on-demand “as-a-service” deployment model to the blockchain ecosystem

  * [[https://chat.hyperledger.org/channel/ci-pipeline|#ci-pipeline]]\\ Used to discuss the Continuous Integration pipeline with respect to [[https://jenkins.hyperledger.org]]

  * [[https://chat.hyperledger.org/channel/composer|#composer]]\\ Used for user discussions on the Hyperledger Composer project

  * [[https://chat.hyperledger.org/channel/composer-ci|#composer-ci]]\\ Used to discuss the Continuous Integration for the Hyperledger Composer project

  * [[https://chat.hyperledger.org/channel/composer-dev|#composer-contributors]]\\ Used for discussions by the contributors/maintainers of the Hyperledger Composer project

  * [[https://chat.hyperledger.org/channel/composer-quality|#composer-quality]]\\ Used for quality assurance and weekly verification test discussion for the Hyperledger Composer project

  * [[https://chat.hyperledger.org/channel/crypto-lib|#crypto-lib]]\\ Used to discuss a shared crypto library across the different Hyperledger projects

  * [[https://chat.hyperledger.org/channel/deployment|#deployment]]\\ Used to discuss deploying any of the Hyperledger projects

  * [[https://chat.hyperledger.org/channel/fabric|#fabric]]\\ Used to discuss the Hyperledger Fabric project

  * [[https://chat.hyperledger.org/channel/fabric-ca|#fabric-ca]]\\ Used to discuss the Certificate Authority of the Hyperledger Fabric project

  * [[https://chat.hyperledger.org/channel/fabric-chaincode-dev|#fabric-chaincode-dev]]\\ Used to discuss the chaincode development system or application chaincode

  * [[https://chat.hyperledger.org/channel/fabric-chaintool|#fabric-chaintool]]\\ Used by the Chaintool project to discuss creating a utility to assist in various phases of Hyperledger Fabric chaincode development, such as compilation, test, packaging, and deployment

  * [[https://chat.hyperledger.org/channel/fabric-crypto|#fabric-crypto]]\\ Used to discuss cryptography for Hyperledger Fabric

  * [[https://chat.hyperledger.org/channel/fabric-dev-env|#fabric-dev-env]]\\ Used to discuss setting up and configuring the developer environment for Hyperledger Fabric
